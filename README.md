# JobNest
Proyecto integrador de la creación de JobNest aplicación wed que ayuda a poder proporcionar y solicitar de trabajos (oficios)

En la actualidad, muchos prestadores independientes no cuentan con un medio formal para mostrar su trabajo, mientras que los usuarios enfrentan dificultades para comparar calidad, precios y confiabilidad.
JobNest surge como una plataforma que centraliza todos estos elementos, permitiendo búsquedas rápidas, comunicación directa y procesos seguros desde el inicio hasta el pago.
